CSS poziciónálás

Static pozició

A static az alapértékmezett pozició. Nem müködnek nála a left, right, top, bottom tulajdonságok

Relative pozició

A relative engedélyezi a left, right, top, bottom tulajdonságok érvényesítését

Absolute pozició

Az absolute is engedélyezi a left, rigth, top, bottom tulajdonságokat, de ő egy abszolút pozociót határoz meg, és nem veszi figyelembe a többi elem helyzetét
